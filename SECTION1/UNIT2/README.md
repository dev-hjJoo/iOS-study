## 1번-프로젝트: 웹 브라우저 만들기
### 학습 내용
* Xcode를 이용하여 화면 구성
  * 툴바, 툴바 버튼, 웹 브라우저, 액티비티 인디케이터
  * 오브젝트 라이브러리 열기는 Command+Shif+L
* 커넥션 인스펙터를 통한 버튼과 컨트롤러 연결
* 화면 정렬 및 구성하기
* 어트리뷰트 인스펙터를 통해 오브젝트 모양 및 색상 변경
### 복습할 문법
